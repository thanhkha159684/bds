Drupal 7 INSTALLATION
------------

1. Copy the gconnect directory to your sites/SITENAME/modules directory or sites/all/modules directory.

2. Enable the module at Administer > Site building > Modules.

3. Visit Administer > Site Configuration > Google Connect Settings to configure the Google Developer API Key.

4. You can get the API key from code.google.com site API console. Please see project page for more information.

5. That's it. :) Visit user/login or user/register page to see Google Connect Buttons.
